# Bike-sales
Analyse des ventes d’une entreprise de vélos – 2025

Ce projet consiste à réaliser une analyse complète des ventes d’une entreprise de distribution d’articles de sport spécialisée dans les vélos, à partir d’un dataset de 113 000 transactions couvrant l’année 2025.

L’objectif est d’exploiter les données commerciales afin d’identifier les tendances de vente, d’évaluer la performance économique et de produire des indicateurs clés d’aide à la décision.
